# L13 作业 · 让 CI 全绿 + MSI 可安装

> 对应教材 L13「交叉编译、MSI 打包与 CI」。代码骨架在 `stages/L13-package/`。

## 需求

1. 完整 CI（三 job：gate / cross-build / windows-package）；
2. WiX 配置，**装完即注册为 Windows 服务**（L12 的服务）；
3. 自签名演示；
4. 产物上传 Release。

骨架里可测的是版本解析 / MSI 命名（补全 `Version::parse` 与 `msi_filename` 的 `todo!()`）；CI 与 WiX 按教材 L13 §2.3/§2.4 的完整示例完成。

## 验收标准

- **CI 全绿**（截图贴 PR）；**Windows 上下载 CI 产出的 MSI，双击安装成功**（`sc query WinMon` 显示 RUNNING）；
- `clippy -D warnings` 通过；**有依赖缓存**（贴加缓存前后的 CI 耗时）；
- **PR 里回答**（D8 评分项）：
  1. CI 跑一次多久？加缓存前后差多少？
  2. 代码里加 `#[allow(dead_code)]` 掩盖警告，CI 会拦住吗？该拦吗？
  3. AI 的第一版 CI，五项检查过了几项？

> 坑：`windows-latest` 不再预装 WiX 3（`choco install wixtoolset`）；`cargo-xwin` 首次下载 SDK 要缓存；MSI 装服务需管理员权限。
